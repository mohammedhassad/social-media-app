## Acknowledgements
<p>Built a full-stack web app with rudimentary features inspired by existing social media platforms such as Facebook and Twitter. The main purpose of this application is to demonstrate how to use the MERN stack technologies to implement features that allow users to connect or follow each other, and interact over shared content.</p>

<br />
<p><strong>Used:</strong> NodeJs, ExpressJs, MongoDB, Mongoose, ReactJs, Material-UI.</p>
